Open Source at MITRE

The MITRE Corporation has been involved with many different open source projects throughout the years, many of which have been founded by MITRE itself. This page is an incomplete list of projects that are available here on GitHub. If you would like to add your project to the list, submit an issue or (better yet) a pull request with the appropriate changes.

MITRE's Technology Transfer Office maintains a list of Open Source projects, many of which are on other hosting environments.

Projects

  • MITREid Connect Reference implementation of the OpenID Connect protocol
  • RHEx RESTful Health Exchange
  • PDMP A testbed to demonstrate integration of prescription drug monitoring programs with health IT systems. Uses hData and OpenSearch technologies.
  • MITRECND cybersecurity
  • iMAS iOS Mobile Application Security
  • hReader Secure patient-centric mobile family health manager
  • popHealth Meaningful Use Clinical Quality Measure reference implementation
  • cypress Meaningful Use Stage 2 Clinical Quality Measure testing and certification tool
  • hQuery A framework for distributed querying of health data
